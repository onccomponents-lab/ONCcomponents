# ONC GitHub-ready Website

Website Next.js cho ONC.

## Chạy local

```bash
npm install
npm run dev
```

## Triển khai

Bạn có thể đưa toàn bộ thư mục này lên GitHub rồi deploy bằng Vercel.

## Ghi chú

- Đã kèm ảnh sản phẩm bạn đã cung cấp trong `public/images/products/`
- Đã kèm file PDF catalog đánh bóng gốc trong `public/images/catalog/`
- Có thể tiếp tục tách thêm ảnh catalog để hiển thị trực tiếp trên web
